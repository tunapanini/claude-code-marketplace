# Claude Code Plugins Marketplace

Claude Code plugins - commands, hooks, and agents by tunapanini

## Installation

### Add Marketplace

```bash
# GitHub 저장소 추가
/plugin marketplace add tunapanini/claude-code-marketplace
```

### Install Plugin

```bash
/plugin  # interactive 방식으로 플러그인(마켓플레이스) 관리함
```
